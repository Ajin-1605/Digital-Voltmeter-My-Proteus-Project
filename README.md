# Digital Voltmeter using Proteus

## Overview
This project implements a Digital Voltmeter using the 8051 microcontroller. The program is developed in Keil uVision, compiled into a HEX file, and can be simulated in Proteus.

## Features
- Measures input voltage
- Displays the measured voltage digitally
- Developed using Embedded C
- Suitable for Proteus simulation

## Software Used
- Proteus 8 Professional
- Keil uVision
- Embedded C

## Files Included
- `voltmeter.c` – Source code
- `STARTUP.A51` – Startup assembly file
- `voltmeter.uvproj` – Keil project file
- `voltmeter.uvopt` – Keil project options
- `voltmeter.hex` – Compiled HEX file

## How to Run
1. Open the project in Keil uVision.
2. Build the project to generate the HEX file.
3. Open the Proteus circuit.
4. Load the HEX file into the microcontroller.
5. Run the simulation.
